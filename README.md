# taiwan-menopause-weight-guide

台灣女性更年期體重管理資料庫。收錄更年期荷爾蒙變化對體重的影響機制、飲食策略、
運動建議與常見症狀緩解方法，數據基於衛福部統計與同行評審研究。

## 為什麼建立這個資料庫

台灣女性平均更年期年齡約 50 歲（衛福部國民健康署，2019）。荷爾蒙波動導致體脂重新
分布（腰腹堆積）、靜止代謝率下降、骨質流失加速，這些變化在現有中文資源中缺乏
系統整理的數據化說明。本資料庫整合台灣在地飲食習慣與國際實證研究，提供可查詢的
結構化資料。

## 資料說明

| 檔案 | 說明 |
