# Contributing to taiwan-menopause-weight-guide

感謝您對本資料庫的興趣。本文件說明如何貢獻資料及哪些貢獻不會被接受。

## 資料來源要求

所有貢獻數據必須來自以下可信來源之一：

- 已發表的同行評審期刊論文（需附 PMID 或 DOI）
- 衛福部國民健康署官方衛教資料
- 衛福部食藥署台灣食品成分資料庫（TFND）
- NAMS（北美更年期學會）立場聲明
- WHO 更年期相關指引

## 如何提交

1. Fork 此 repository
2. 新增或修改 JSON 檔案內容
3. 確保每筆資料包含 `source` 欄位（具體來源或 PMID）
4. 更新 `_meta.last_updated`（格式：YYYY-MM-DD）
5. 提交 Pull Request，說明修改原因和來源

## 接受的貢獻

- 新增有文獻支持的更年期飲食和症狀管理策略
- 修正現有數據錯誤（需附來源）
- 台灣在地化補充（台灣食材、就醫資源）
- 更新過時的建議（需附更新文獻）

## 不接受的貢獻

- **荷爾蒙補充療法（HRT）具體建議**：屬於醫療處方範疇，超出本資料庫範圍
- **特定保健食品品牌推薦**：商業利益衝突
- **草藥/中藥療法**：若無嚴格 RCT 支持（只有個案或傳統用途）
- **誇大效果**：「完全消除熱潮紅」、「完全阻止骨質流失」等不實描述
- **無來源數據**：任何沒有 PMID 或官方機構來源的健康聲明

## 資料格式

每筆資料必須包含：
- `id`：唯一識別碼（kebab-case）
- `source`：具體來源名稱 + 年份 + PMID（若為期刊論文）
- `_meta.last_updated`：最後更新日期
